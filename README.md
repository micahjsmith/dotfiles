# dotfiles

Micah's dotfiles: bashrc, vimrc, etc.

- License: MIT License

## Setup a new box

```
./setup.py
```

## Archive dotfiles

```
./archive.sh
```

## Unarchive dotfiles

```
cd path/to/archive/directory
./unarchive.sh
```
